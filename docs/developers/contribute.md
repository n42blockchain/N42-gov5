# Contribute

N42 welcomes contributions from the community! Whether you're fixing bugs, adding features, or improving documentation, your help is appreciated.

## Getting Started

1. Fork the repository on GitHub
2. Clone your fork locally
3. Create a new branch for your changes
4. Make your changes and test them
5. Submit a pull request

## Development Setup

```bash
# Clone the repository
git clone https://github.com/n42blockchain/N42
cd N42

# Build the project
make build

# Run tests
make test
```

## Code Style

- Follow Go's official style guidelines
- Run `gofmt` before submitting
- Run `go vet` and fix any warnings
- Write meaningful commit messages

## Documentation

- Update documentation when changing APIs
- Add doc comments for public functions
- Include examples where helpful

## Testing

- Write unit tests for new functionality
- Ensure all existing tests pass
- Add integration tests for complex features

## Pull Request Process

1. Update the CHANGELOG if applicable
2. Ensure CI passes
3. Request review from maintainers
4. Address feedback promptly

## Getting Help

- Join our [Telegram](https://t.me/N42_official)
- Open an issue on GitHub
- Check existing documentation

Thank you for contributing to N42!
